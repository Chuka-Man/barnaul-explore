
# Barnaul Explore

Учебный frontend-проект, посвящённый городу Барнаулу.


## Технологии

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Responsive Design

## Возможности

* адаптивная верстка;
* навигация по странице;
* карточки достопримечательностей;
* hover-анимации;
* семантическая HTML-разметка.

## Достопримечательности

* Нагорный парк
* Проспект Ленина
* Речной вокзал
